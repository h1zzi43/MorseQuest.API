# MorseQuest - Квест и переводчик азбуки Морзе

Веб-приложение для изучения азбуки Морзе через переводчик, звуковые сигналы, тренировку и мини-квест.

## Стек технологий

- Backend: ASP.NET Core 10.0 (C#)
- Database: PostgreSQL
- Frontend: HTML, CSS, JavaScript
- API: REST API

## Системные требования

- .NET 10.0 SDK
- PostgreSQL 14+
- Современный браузер (Chrome, Firefox, Edge)
