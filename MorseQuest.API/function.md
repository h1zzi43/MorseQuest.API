#  Morse Code Translator  — Полный список реализованных функций

> **Проект:** Visual Studio 2026  
> **Тема:**  Переводчик азбуки Морзе 
> **Версия:** 1.0.0  

---

##  Реализованные функции

###  Переводчик Морзе
- Перевод текста (латиница и кириллица) в азбуку Морзе
- Перевод кода Морзе в текст
- Поддержка букв, цифр и специальных символов
- Быстрые примеры для демонстрации
- Копирование результата в буфер обмена
- История переводов (сохраняется в `localStorage`)

###  Звук Морзе
- Генерация WAV аудио с азбукой Морзе
- Короткий сигнал для точки (.)
- Длинный сигнал для тире (−)
- Паузы между символами и буквами
- Кнопка прослушивания для любого текста

### 🎮 Тренировка
- Случайная генерация символов Морзе
- Выбор правильной буквы из 4 вариантов
- Три уровня сложности:
  - Легкий
  - Средний
  - Сложный
- Подсчет правильных и неправильных ответов
- Система серий (streak)
- Прогресс-бар прохождения
- Сохранение имени пользователя
- Отображение результатов после завершения

###  Мини-квест
- 5 заданий разного типа
- Ввод текстового ответа
- Выбор из вариантов
- Таймер прохождения
- Подсчет очков с бонусами за скорость и точность
- Отображение итоговых результатов

###  Таблица лидеров
- Раздельные рейтинги для:
  - Тренировки
  - Квеста
- Топ-10 игроков
- Отображение очков и даты
- Сохранение результатов в базу данных

##  API Endpoints

| Метод | Endpoint | Описание |
|-------|----------|----------|
| **POST** | `/api/morse/translate` | Перевод текста |
| **POST** | `/api/morse/audio` | Генерация аудио |
| **GET** | `/api/morse/training/question` | Получение вопроса |
| **POST** | `/api/morse/training/check` | Проверка ответа |
| **POST** | `/api/morse/results/save` | Сохранение результата |
| **GET** | `/api/morse/leaderboard` | Таблица лидеров |
| **GET** | `/api/morse/quest/tasks` | Задания квеста |
| **POST** | `/api/morse/quest/check` | Проверка ответа квеста |
